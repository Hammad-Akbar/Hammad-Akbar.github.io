PennyLogger (2025)

iOS app to log daily expenses, view history, and analyze spending.

  • iPhone app built with SwiftUI to track daily expenses and spending patterns.
  • Supports Apple ID login, swipeable daily view, image attachments, and Core Data storage.
  • Features include: sorting by amount/time, detailed expense view, iCloud backup, photo uploads and currency conversions using the ExchangeRate-API.
  • Custom filtering and expense exporting.
  • Designed with responsive UI, persistent data, and accessibility support in mind.
