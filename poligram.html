Poligram (2020)

Learn about American politics, take ideology quizzes, participate in public polls, and explore the political geography of the United States of America.

Visit Project

  • Built with engineers from Bloomberg and Facebook using Scrum and Agile methodologies.
  • Led as Project Manager and Lead Software Engineer.
  • Tech stack: Python-Flask, ReactJS, JavaScript, HTML, CSS, PostgreSQL, Git, Heroku.
  • Automated testing with Circle CI, OAuth login via AWS EC2.
  • Used React Router, Socket.IO, News API, Dictionary API.
  • Created JSON data for map details and ideology quiz.
  • Database design stores user login state and feedback.
