Food Galore (2019)

Find recipes, nutrition info, and live tweets about food.

Visit Project

  • Web app to explore recipes, ingredients, nutrition, and social mentions.
  • Built using Python-Flask, HTML, CSS, Git, Heroku, and APIs via AWS EC2.
  • Spoonacular API used for nutrition facts and recipe info.
  • Tweepy API integrated to fetch real-time tweets about the food.
  • Developed custom HTML divisions to structure content clearly.
