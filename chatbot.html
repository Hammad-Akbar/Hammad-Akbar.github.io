The Chat Box (2019)

A real-time chat app with image, GIF, and bot command support.

Visit Project

  • Built to enable users to chat, send images/GIFs, and use bot commands.
  • Tech stack: Python-Flask, ReactJS, JavaScript, HTML, CSS, PostgreSQL, Git, Heroku.
  • Automated unit testing and OAuth login via AWS EC2.
  • React Socket.IO integration for real-time messaging.
  • Used Funtranslate and Chuck Norris APIs for bot features.
  • Database design stores user names and chat history for new users.
